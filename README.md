# Project RAG

Progress 1 — Setup Infrastruktur Dasar

## Instalasi Tools
Semua tools berikut telah berhasil diinstall pada Windows:
- Docker Desktop
- Node.js (LTS)
- n8n (local)
- VSCode
- Git & GitHub
- Ngrok

## Cara Menjalankan n8n (Docker Compose)
1. Masuk ke folder `infra/`
2. Jalankan: `docker compose up -d`
3. Akses n8n di: http://localhost:5678

## Struktur Project
Lihat folder pada repository:
- `/infra` → file docker-compose & config
- `/docs` → laporan & screenshot
- `/webapp` → untuk progress 3
- `/n8n` → export workflow n8n (progress 2 & 3)
